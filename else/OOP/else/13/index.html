数値データの可視化

Chart.jsの概要

Chart.jsは、データ可視化のためのフリーでオープンソースのJavaScriptライブラリです。複数タイプ（棒、折れ線、面積、円、バブル、レーダ、散布図など）のグラフ形式をサポートしています。これは、2013年に作成され、現在はコミュニティによってメンテナンスされています。Chart.jsは、HTML5 canvasでレンダリングできるデータ可視化ライブラリとして広く採用されています。（引用：「Wikipedia：Chart.js」）

グラフ（折れ線、レーダ）描画例

※ Chart.js-Getting Started 等からライブラリファイル（chart.js※直接リンク）取得

ChartJS_Sample.html

<html>
<head>
<meta charset="UTF-8">
 
<!-- グラフ描画ライブラリ読み込み -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>
 
<body>
<!-- 描画領域 -->
<div style="width:400px; text-align:center">
<canvas id="mycanvas" width="400" height="300"></canvas>
<h2>図：グラフ描画（例）</h2>
</div>
 
<script>

 // グラフ描画スクリプト（chart.js利用）

 var ctx = document.getElementById('mycanvas');
 
var myChart = new Chart(ctx, {

   type: 'line',   // 折れ線グラフ描画

 //  type: 'radar',  // レーダーチャート描画

   data: {         // データセット作成（グラフ種別依存）

     labels: ['A', 'B', 'C', 'D'],

     datasets: [

     {

       label: '赤',

       data: [40, 30, 20, 45],

       borderColor: 'red',

     }, {

       label: '緑',

       data: [ 5, 25, 35, 15],

       borderColor: 'green',

     }, {

       label: '青',

       data: [45, 5, 40, 10],

       borderColor: 'blue',

     } ],

   },

   options: {      // 表示設定（グラフ種別依存）

     scales: {

       y: {

         min: 0,

         max: 50,

         title: {

           display: true,

           text: 'Y-axis',

         },

       },

       x: {

         title: {

           display: true,

           text: 'X-axis',

         },

       },

     },

   },

 });
</script>
 
</body>
</html>
