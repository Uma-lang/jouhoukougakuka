PHP 入門

PHP は、オープンソースの汎用スクリプト言語（正式名称 PHP:Hypertext Preprocessor）です。特に、サーバサイドで動作する Web アプリケーションの開発に適しています。言語構造は、簡単で理解しやすく、Ｃ言語の基本構文に基づいてます。 手続き型のプログラミングに加え、オブジェクト指向のプログラミングもできます。Linux や FreeBSD 等の多くの Unix系システム、Microsoft Windows、Mac OS X など、主要な OS で動作します。 また、Apache や Microsoft IIS をはじめとした多くのウェブサーバをサポートしています。オープンソースなライセンスでリリースされているため、商用・個人用途を問わず自由に使用することができます。各種ライブラリも充実しており、さまざまなWeb サイト、WebアプリケーションがPHPを使って開発されています。（日本PHPユーザ会のホームページ「PHP とは」より引用）

サーバサイドで処理するタイプのスクリプト言語（PHP）で運用されるシステムは、クライアント側に負荷をかけることがなく、その処理内容を非公開にできる、という特長があります。その反面、クライアントからの処理要求が過密になると、サーバシステムが抱える負担も過大になるので、応答時間が長く（反応が遅く）なります。

防衛大学校校内の個人用ホームページを管理しているウェブサーバ（home.nda.ac.jp）は、PHP 言語によるサーバサイド処理に対応しています。そのため、個人用ホームページ領域（ホームディレクトリ/public_html/ 以下）に PHP で記述されたファイル（拡張子「.php」）を配置するだけで、PHP によるサーバサイド処理型 Web アプリケーションを公開できます。

本講義では、以下に示す簡単なサンプルプログラムを実際に入力（ファイルを作成してホームページ領域で公開）し、PHP のプログラミングスタイル（アルゴリズムなどの複雑なプログラミング教育は実施しない）を学びます。

以下、出力データ（表示内容のソース）は講義資料のページ上に示さないので、各プログラムコードをよく読み、出力データがどのようになるかイメージできた段階で処理（サーバサイド）を実行し、ウェブページのソースを確認（ブラウザ表示上で右クリック「ページのソースを表示」など）するよう心がけましょう。




ページソース確認法（ブラウザ表示上で右クリック）


とにかく使ってみよう！

sample_info.php

<html>
<body>
<?php
  phpinfo();
?>
</body>
</html>

※phpinfo()は、PHPの設定情報を出力する関数の記述例（引数なしの場合、INFO_ALL）

文字を表示してみる（echo）

sample_echo.php

<html>
<body>
<?php
echo "<h1>\n";
echo "ABC<br>\n","DEF<br>\n";
echo "</h1>\n";
?>
</body>
</html>
参考：エスケープシーケンス（制御コード、特殊文字）の記述例

文字列の連結（演算）

sample_concat.php

<html>
<body>
<?php
$a = "ABC";
$b = "DEF";
$c = $a.$b;
echo $a,"<br>\n";
echo $b,"<br>\n";
echo $c,"<br>\n";
?>
</body>
</html>

変数を用いた演算

sample_plus.php

<html>
<body>
<?php
$a = 3;
$b = 7;
echo $a+$b,"<br>\n";
?>
</body>
</html>

条件分岐（if文）

sample_if.php

<html>
<body>
<?php
$select = "B";
if($select == "A"){
  echo "-- A --\n";
}else if($select == "B"){
  echo "-- B --\n";
}else if($select == "C"){
  echo "-- C --\n";
}else{
  echo "-- ? --\n";
}
?>
</body>
</html>

反復処理（while文）

sample_while.php

<html>
<body>
<?php
$i = 0;
while($i < 10){
  $i++;
  echo "i=".$i."<br>\n";
}
?>
</body>
</html>

配列変数の利用（array, foreach）

sample_foreach.php

<html>
<body>
<?php
$a = array(1,3,5,2,4,6,999);
foreach($a as $i){
  echo "i=".$i."<br>\n";
}
?>
</body>
</html>
