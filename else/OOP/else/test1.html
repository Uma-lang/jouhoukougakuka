シラバス対策

  1. クライアントがWebサーバに対してリクエストを送り、Webサーバがそのリクエストに応じて必要なデータやWebページをレスポンスとして返す、という通信モデル。

  2. 構造:ウェブページの構造やコンテンツを記述するためのHTML、ウェブページのスタイルやデザインを定義するためのCSS、ウェブページでの動的な動作やユーザーとのやり取りを実現するJavaScriptなどからなる。
    役割:Webサーバーから取得したHTML、CSS、JavaScriptなどのファイルを解釈し、ユーザーが閲覧できる形で画面に表示することである。ブラウザはこれらのファイルを解析してWebページをレンダリング（描画）し、ユーザーの操作に応じてページを更新する。

  4. Webページにおいてユーザーのブラウザ上で実行される処理。JavaScriptなどのスクリプト言語を使用して、ユーザーとのインタラクション、入力データの検証、ページの動的な更新などを行う。これにより、サーバーに負荷をかけずにリアルタイムな応答性を実現できる。

  5. サーバシステムは、クライアントからのリクエストに応じてサービスやデータを提供するためのハードウェアとソフトウェアの集合。主な構成要素として以下が挙げられる。

    • ウェブサーバ:HTTPプロトコルを用いてウェブページやリソースを配信する。
    • アプリケーションサーバ:ビジネスロジックを実行し、動的なコンテンツ生成やデータ処理を行う。
    • データベースサーバ:データの保存、管理、検索を担当する。
    • ファイアウォール:ネットワークセキュリティを維持し、不正なアクセスや攻撃からシステムを保護する。
    • ロードバランサ:複数のサーバ間でトラフィックを分散し、システム全体の性能と信頼性を向上させる。

  6. ウェブサーバと外部のアプリケーションプログラムを連携させるための標準的なインターフェース。

  7. ウェブアプリケーションにおいてサーバ側で実行される処理のこと。ユーザーからのリクエストを受け取り、データベースのアクセス、ビジネスロジックの実行、動的なコンテンツの生成などを行う。これにより、ユーザーに対して適切な情報やサービスを提供することが可能となる。

  8. ページ記述言語は、ウェブページの構造やデザインを定義するために使われる。PHPはサーバサイドのスクリプト言語であり、動的にページ記述言語のコードを生成・操作するために利用される。PHPプログラムはサーバ上で実行され、その結果として生成されたHTMLやCSSがクライアントに送信される。つまり、PHPはページ記述言語を動的に生成・管理するための手段として機能し、動的なウェブコンテンツを提供する関係にある。

  13. プログラム内で変数が有効で参照可能な範囲のこと。スコープには主にグローバルスコープとローカルスコープがあり、グローバルスコープの変数はプログラム全体からアクセス可能である。一方、ローカルスコープの変数は特定の関数やブロック内でのみ有効で、その外部からはアクセスできない。

  14. プログラムをオブジェクトと呼ばれる独立した部品の集合として設計・構築する考え方。各オブジェクトはデータ（属性）と、それを操作する手続き（メソッド）を持つ。カプセル化、継承、ポリモーフィズムなどの特性を利用して、コードの再利用性や保守性を高めることが可能となる。

  15. クラスとは、オブジェクトを生成するための設計図やテンプレート。クラスには属性（データ）とメソッド（操作）が定義されており、これを基にオブジェクト（インスタンス）が作られる。オブジェクトはクラスの具体的な実体であり、クラスで定義された属性やメソッドを持つ。つまり、クラスが設計図で、オブジェクトがその設計図から作られた建物のような関係にある。

  18. オブジェクト指向プログラミングにおいて、データ（属性）とそれを操作するメソッド（手続き）を一つのオブジェクト内にまとめ、外部からの直接的なアクセスを制限する概念。これにより、内部実装を隠蔽し、オブジェクト間の相互作用を明確に定義できる。結果として、コードの保守性や再利用性が向上し、バグの発生を減少させることが可能となる。 

  19.                         <head>
                        <?php
                        class MyArray {
                          public $data = array();          // スーパークラス（親クラス）
                          function MyArray($new_data)
                          {
                            $this->data = $new_data;
                          }
                          function DumpArray()
                          {
                            $data = $this->data;
                            for($i=0; $i<count($data); $i++){
                              print "$i: $data[$i]<br>";
                            }
                          }
                        }
                        class ExArray extends MyArray {    // サブクラス（子クラス）
                          function ExArray($new_data)
                          {
                            parent::MyArray($new_data);    // スーパークラスのコンストラクタ利用
                          }                                // ※ "::"はネームスペース指定の記述
                          function SortArray()             // サブクラスでメソッド（機能）を追加
                          {
                            sort($this->data);             // 配列の並べ替え
                          }
                        }
                        ?>
                        </head>
                        
                        <body>
                        <?php
                        $a = array('a1','b2','a3','b4','a5');       // サブクラスを用いてオブジェクト生成
                        $obj = new ExArray();
                        $obj->DumpArray();                 // スーパークラスのメソッド実行
                        $obj->SortArray();                 // サブクラスで追加したメソッド実行
                        $obj->DumpArray();                 // スーパークラスのメソッド実行
                        ?>
                        </body>
                        

  20.                         <head>
                        <?php
                        abstract class MyArray              // スーパークラス（親クラス）
                        {
                          public $data = array();
                          function MyArray($new_data)
                          {
                            $this->data = $new_data;
                          }
                          function DumpArray()
                          {
                            $data = $this->data;
                            for($i=0; $i<count($data); $i++){
                              print "$i: $data[$i]<br>";
                            }
                          }
                          // 抽象メソッド（拡張クラスでの定義を強制 ※空の状態）
                          abstract function SortArray();
                        }
                        class ExArrayA extends MyArray      // サブクラス（子クラス）
                        {
                          function ExArrayA($new_data)
                          {
                            parent::MyArray($new_data);
                          }
                          function SortArray()              // 抽象メソッド定義（必須）
                          {
                            sort($this->data);              // データ並べ替え（昇順）
                          }
                        }
                        class ExArrayR extends MyArray      // サブクラス（子クラス）
                        {
                          function ExArrayR($new_data)
                          {
                            parent::MyArray($new_data);
                          }
                          function SortArray()              // 抽象メソッド定義（必須）
                          {
                            Shuffle($this->data);           // データ並べ替え（ランダム）
                          }
                        }
                        ?>
                        </head>
                        
                        <body>
                        <?php
                        $a = array('a1','b2','a3','b4','a5');
                        
                        $objA = new ExArrayA($a);
                        $objA->DumpArray();
                        $objA->SortArray();                 // 拡張定義した抽象メソッド実行
                        $objA->DumpArray();
                        
                        $objR = new ExArrayR($a);
                        $objR->DumpArray();
                        $objR->SortArray();                 // 拡張定義した抽象メソッド実行
                        $objR->DumpArray();
                        ?>
                        </body>
