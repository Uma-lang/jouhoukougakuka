JavaScriptの innerText プロパティ

innerText プロパティは、HTML 要素内の表示されているテキストを取得したり、変更したりするために使用されます。このプロパティを使うと、タグの中身のテキストを簡単に操作できます。

例 1: テキストを取得する

以下の例では、ボタンをクリックすると、id="example" を持つ要素のテキストを取得して表示します。

<h1 id="example">このテキストを取得します</h1>
<button onclick="showText()">テキストを取得する</button>

<script>
function showText() {
    var element = document.getElementById("example");
    alert(element.innerText);  // 要素のテキストをアラートで表示
}
</script>

実際に試してみましょう:

このテキストを取得します

例 2: テキストを変更する

次に、innerText を使って要素内のテキストを変更する例です。

<h1 id="example2">このテキストを変更します</h1>
<button onclick="changeText()">テキストを変更する</button>

<script>
function changeText() {
    var element = document.getElementById("example2");
    element.innerText = "テキストが変更されました！";  // テキストを変更
}
</script>

実際に試してみましょう:

このテキストを変更します

innerText と textContent の違い

innerText と textContent は似ていますが、いくつかの違いがあります。innerText は表示されているテキストのみを取得・設定し、スタイルに依存します。一方、textContent はスタイルに関係なく、要素内の全てのテキストを取得します。

textContent の例:

<div id="hidden" style="display:none;">このテキストは非表示です。</div>
<button onclick="showContent()">textContent を取得する</button>

<script>
function showContent() {
    var element = document.getElementById("hidden");
    alert(element.textContent);  // 要素の全テキストを表示（非表示部分も含む）
}
</script>

実際に試してみましょう:

このテキストは非表示です。
