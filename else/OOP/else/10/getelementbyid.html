JavaScriptの getElementById メソッド

getElementById は、JavaScript で HTML 要素にアクセスするためのメソッドです。このメソッドを使って、特定の id 属性を持つ要素を取得し、操作することができます。以下に基本的な使い方と例を示します。

基本的な構文

document.getElementById("id名")

この構文を使うことで、HTML ドキュメント内の指定された id を持つ要素にアクセスし、その要素を操作することができます。

例 1: テキストを変更する

以下の例では、ボタンをクリックすると id="example" を持つ要素のテキストが変更されます。

<h1 id="example">このテキストを変更します</h1>
<button onclick="changeText()">テキストを変更する</button>

<script>
function changeText() {
    var element = document.getElementById("example");
    element.innerText = "テキストが変更されました！";
}
</script>

実際に試してみましょう:

このテキストを変更します

例 2: 要素のスタイルを変更する

次に、getElementById を使って要素のスタイルを変更する例です。以下のコードでは、ボタンをクリックすると、id="example2" を持つ要素の色が赤に変わります。

<h1 id="example2">このテキストの色を変えます</h1>
<button onclick="changeColor()">色を赤に変更する</button>

<script>
function changeColor() {
    var element = document.getElementById("example2");
    element.style.color = "red";
}
</script>

実際に試してみましょう:

このテキストの色を変えます

その他の操作例

  • 要素を非表示にする: document.getElementById("myElement").style.display = "none";
  • 画像を変更する: document.getElementById("myImage").src = "new_image.jpg";
  • フォームの入力値を取得する: var value = document.getElementById("inputField").value;

注意点

id 属性は HTML ドキュメント内で一意である必要があります。つまり、同じ id を複数の要素に使わないようにしてください。
