selectedIndex の説明

selectedIndex は、HTML の <select> 要素に関連する JavaScript のプロパティです。ユーザーが選択したオプションのインデックス（番号）を取得したり、指定することができます。このプロパティは、選択されたオプションの インデックス番号を操作するために使います。

selectedIndex プロパティの用途

selectedIndex を使用することで、以下のような操作が可能です:

  • 選択されているオプションのインデックス番号を取得
  • 特定のオプションをプログラム的に選択

使用例

1. 選択されたオプションのインデックスを取得する例

次の例では、ユーザーが選択したオプションのインデックス番号を取得し、アラートで表示します。

<select id="mySelect" onchange="showSelectedIndex()">
    <option value="1">オプション 1</option>
    <option value="2">オプション 2</option>
    <option value="3">オプション 3</option>
    </select>

<script>
function showSelectedIndex() {
    var selectElement = document.getElementById("mySelect");
    var index = selectElement.selectedIndex;  // 選択されたオプションのインデックスを取得
    alert("選択されたオプションのインデックス: " + index);
}
</script>

2. 特定のオプションを選択する例

次の例では、ボタンをクリックすることで、プログラムから 3 番目のオプションを自動的に選択することができます。

<select id="mySelect">
    <option value="1">オプション 1</option>
    <option value="2">オプション 2</option>
    <option value="3">オプション 3</option>
    </select>

<button onclick="selectOption()">3番目のオプションを選択</button>

<script>
function selectOption() {
    var selectElement = document.getElementById("mySelect");
    selectElement.selectedIndex = 2;  // 3 番目のオプションを選択
}
</script>

selectedIndex プロパティの動作

selectedIndex は、インデックス番号を返すプロパティです。最初のオプションのインデックスは 0 で始まり、次のようにカウントされます:

  • 1番目のオプション → インデックス 0
  • 2番目のオプション → インデックス 1
  • 3番目のオプション → インデックス 2

特定のインデックス番号を指定することで、プログラムから自動的にオプションを選択状態にできます。また、選択がない場合、selectedIndex の値は -1 となります。
