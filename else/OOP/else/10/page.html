page() 関数の使い方

page() 関数は、JavaScript を使ってページを別の URL に遷移させるために使用されます。以下のボタンをクリックすると、https://www.example.com にリダイレクトされます。

pageContent() 関数でページ内容を動的に変更する例

次に、pageContent() 関数を使用して、ページの一部の内容を動的に変更する例です。以下のボタンをクリックすると、テキストが動的に変更されます。

ここに表示されているテキストは変更前の内容です。

コード例

1. ページ遷移のための page() 関数

<script>
function page() {
    location.href = "https://www.example.com";  // 指定されたURLにページを遷移
}
</script>

<button onclick="page()">ページを遷移</button>

2. ページ内容を変更する pageContent() 関数

<script>
function pageContent() {
    document.getElementById("content").innerHTML = "ページの内容が変更されました！";
}
</script>

<button onclick="pageContent()">ページ内容を変更</button>

<div id="content">ここに表示されているテキストは変更前の内容です。</div>

まとめ

このように、page() や pageContent() 関数を使うことで、ページのリダイレクトや、ページ内容の動的な変更を実現できます。ユーザーの操作に基づいてこれらを実行できるため、インタラクティブなウェブサイトを構築する際に非常に便利です。
