onchange イベントの説明

onchange イベントは、フォーム要素（<input>、<select>、<textarea> など）の内容が変更されたときに発生するイベントです。このイベントは、選択や入力が完了した後に発生し、JavaScript 関数を呼び出して処理を行うことができます。

使用例

1. ドロップダウンメニューの選択が変更されたとき

ユーザーが選択した値をアラートで表示します。

<select id="mySelect" onchange="showSelectedValue()">
    <option value="1">オプション 1</option>
    <option value="2">オプション 2</option>
    <option value="3">オプション 3</option>
</select>

<script>
function showSelectedValue() {
    var selectElement = document.getElementById("mySelect");
    var selectedValue = selectElement.value;
    alert("選択された値は: " + selectedValue);
}
</script>

2. テキスト入力が完了したとき

ユーザーが入力したテキストをアラートで表示します。

<input type="text" id="myInput" onchange="showInputValue()" placeholder="名前を入力してください">

<script>
function showInputValue() {
    var inputElement = document.getElementById("myInput");
    alert("入力された値は: " + inputElement.value);
}
</script>

3. チェックボックスの状態が変更されたとき

チェックボックスがチェックされたかどうかをアラートで表示します。

チェックボックス 
<input type="checkbox" id="myCheckbox" onchange="showCheckboxStatus()"> チェックボックス

<script>
function showCheckboxStatus() {
    var checkbox = document.getElementById("myCheckbox");
    if (checkbox.checked) {
        alert("チェックされました");
    } else {
        alert("チェックが外されました");
    }
}
</script>

まとめ

onchange イベントは、フォーム要素の内容が変更されたときに発生し、その変更に基づいて処理を実行するために使用されます。ユーザーの操作に応じて適切な反応を示すインタラクティブなウェブページを作成する際に非常に役立ちます。
