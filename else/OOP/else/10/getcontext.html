Canvas の getContext('2d') の使い方

getContext('2d') は、HTML5 の <canvas> 要素を使用して 2D 描画を行うためのメソッドです。2D コンテキストを取得することで、図形やテキストをキャンバスに描くことができるようになります。

基本的な使い方

以下の手順で、キャンバスに描画するためのコンテキストを取得し、図形やテキストを描画できます。

  1. まず、HTML 内に <canvas> 要素を作成します。
  2. JavaScript で、getContext('2d') メソッドを使用して 2D 描画コンテキストを取得します。
  3. 取得したコンテキストを使って、図形やテキストを描画します。

コード例

<canvas id="myCanvas" width="300" height="150"></canvas>

<script>
    var canvas = document.getElementById("myCanvas");
    var ctx = canvas.getContext('2d');

    // 四角形を描画
    ctx.fillStyle = "lightblue";
    ctx.fillRect(10, 10, 150, 100);

    // 四角形の枠線を描画
    ctx.strokeStyle = "blue";
    ctx.lineWidth = 5;
    ctx.strokeRect(10, 10, 150, 100);

    // テキストを描画
    ctx.fillStyle = "black";
    ctx.font = "20px Arial";
    ctx.fillText("Hello Canvas", 20, 60);
</script>

まとめ

このように、getContext('2d') メソッドを使うことで、HTML の <canvas> 要素上にさまざまな図形やテキストを描画できます。2D コンテキストを操作することで、アニメーションやインタラクティブなグラフィックスも実現可能です。
