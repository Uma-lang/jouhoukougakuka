JavaScriptのdocumentオブジェクトとは

document は、JavaScriptにおいて、ウェブページの HTMLドキュメント全体 を表すオブジェクトです。document オブジェクトを使用することで、ページの構造や内容を動的に操作することができ、ユーザーとのインタラクションを管理することができます。これにより、ウェブページをより動的でインタラクティブにします。

documentオブジェクトでできること

  • HTML要素にアクセスする: ページ内の特定の HTML 要素を選択して操作します。
  • ページの内容を変更する: ページに新しい要素を追加したり、内容を変更したりします。
  • イベントを処理する: ユーザーの操作（クリックや入力など）に応じてイベントを処理できます。

具体的な使い方とメソッド

1. HTML要素へのアクセス

様々なメソッドを使って、HTML 要素にアクセスできます。

// IDを使って要素を取得
var element = document.getElementById("myElement");

// クラス名で要素を取得
var elements = document.getElementsByClassName("myClass");

// CSSセレクタを使って最初の要素を取得
var element = document.querySelector(".myClass");

2. ページの内容を変更する

ページの内容を動的に変更できます。

// 要素のHTML内容を変更
document.getElementById("myElement").innerHTML = "新しい内容";

// 要素のテキストだけを変更
document.getElementById("myElement").textContent = "新しいテキスト";

3. 新しい要素を追加する

JavaScript を使って新しい要素をページに追加することも可能です。

// 新しい段落を作成して追加
var newElement = document.createElement("p");
newElement.textContent = "この段落は動的に追加されました。";
document.body.appendChild(newElement);

4. ページの情報を取得する

現在のページに関する情報を取得することができます。

// ページのタイトルを取得・設定
console.log(document.title);  // 現在のタイトルを出力
document.title = "新しいタイトル";  // タイトルを変更

// 現在のURLを取得
console.log(document.URL);

5. イベントリスナーを追加する

ユーザーの操作に応じてイベントを処理することができます。

document.getElementById("myButton").addEventListener("click", function() {
    alert("ボタンがクリックされました！");
});

document の主なプロパティとメソッド

  • プロパティ: document.title, document.URL, document.body など。
  • メソッド: getElementById(), getElementsByClassName(), querySelector(), createElement() など。

document を使った例

1. ページタイトルを変更

document.title = "新しいページタイトル";

2. IDを持つ要素にテキストを挿入

document.getElementById("myElement").textContent = "新しいテキスト";

3. ボタンクリック時にアラートを表示

document.getElementById("myButton").addEventListener("click", function() {
    alert("ボタンがクリックされました！");
});

まとめ

document オブジェクトは、JavaScript で HTML ドキュメント全体を操作するための強力なツールです。ページの内容の動的な変更や、ユーザーの操作に応じたイベント処理が可能で、インタラクティブなウェブページを作成するために欠かせない要素です。
