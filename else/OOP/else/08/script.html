<script> タグとは

<script> タグは、HTML ドキュメント内で JavaScript を埋め込む、または外部の JavaScript ファイルを読み込むために使用されます。このタグは、Webページにインタラクティブな要素や動的な機能を追加するために不可欠です。

<script> タグの基本構造

1. インラインで JavaScript を記述する場合

<script>
  // JavaScriptのコード
  console.log("Hello, World!");
</script>

このように、<script> タグ内に直接 JavaScript を記述すると、ブラウザが HTML を解析する際にこのスクリプトが実行されます。

2. 外部ファイルの JavaScript を読み込む場合

<script src="myscript.js"></script>

外部 JavaScript ファイルを読み込む場合は、src 属性を使用してファイルのパスを指定します。これにより、複数のページで同じスクリプトを使いまわせるため、コードの管理が簡単になります。

<script> タグの属性

  • src: 外部 JavaScript ファイルを指定するための属性。
  • async: スクリプトを非同期に読み込み、他の HTML の読み込みをブロックしないようにします。
  • defer: スクリプトの実行を HTML 解析の完了後まで遅延させます。

例：外部ファイルのスクリプトを非同期に読み込む

<script src="myscript.js" async></script>

<script> タグの配置

1. <head> タグ内に配置

<head> タグ内にスクリプトを配置すると、ページのコンテンツが読み込まれる前に実行されます。ただし、DOM 操作は失敗することがあります。

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log("ヘッド内のスクリプト");
    </script>
</head>
<body>
    <h1>Hello, World!</h1>
</body>
</html>

2. <body> タグの最後に配置

<body> タグの最後に配置すると、HTML コンテンツの読み込みが完了してからスクリプトが実行されるため、DOM 操作が確実に行えます。

<!DOCTYPE html>
<html>
<head>
    <title>Sample Page</title>
</head>
<body>
    <h1>Hello, World!</h1>

    <script>
        console.log("ボディ終了前のスクリプト");
    </script>
</body>
</html>

スクリプトの読み込みと実行順序

  • 同期的なスクリプトの読み込み: デフォルトでは、ブラウザは <script> タグに遭遇すると、その場でスクリプトを読み込んで実行し、HTML の解析を一時停止します。
  • 非同期読み込み（async）: async 属性を使うと、スクリプトが他のコンテンツの読み込みをブロックせず、非同期に読み込まれます。
  • 遅延読み込み（defer）: defer 属性を使うと、HTML の解析が完了した後にスクリプトが実行されます。複数のスクリプトがある場合、読み込み順に実行されます。

例: <script> タグの使用方法

1. インライン JavaScript の例

<!DOCTYPE html>
<html>
<head>
    <title>インラインJavaScriptの例</title>
</head>
<body>
    <h1>インラインJavaScriptの実行</h1>
    <script>
        document.write("このテキストはJavaScriptによって挿入されました。");
    </script>
</body>
</html>

2. 外部ファイルの JavaScript の例

外部ファイル「script.js」の内容:

document.write("これは外部ファイルからのJavaScriptです。");

HTMLファイル:

<!DOCTYPE html>
<html>
<head>
    <title>外部JavaScriptの例</title>
</head>
<body>
    <h1>外部JavaScriptの実行</h1>
    <script src="script.js"></script>
</body>
</html>

まとめ

  • <script> タグは、JavaScript を HTML ページに埋め込むためのタグです。
  • src 属性で外部 JavaScript ファイルを指定できます。
  • async や defer 属性で、スクリプトの読み込みと実行のタイミングを制御できます。
  • スクリプトの配置によって、ページの読み込み速度や動作に影響を与えます。
