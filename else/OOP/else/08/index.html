JavaScript 入門

JavaScriptは、WebブラウザNetscape Navigator（ネットスケープコミュニケーションズ社製のブラウザ）に起源を持ち、現在では様々なクライアントサイド処理に利用されているオブジェクト指向型のプログラミング言語の一つです。JavaScriptは、HyperText Markup Language（HTML、主に文書構造を記述）やCascading Style Sheets（CSS、文書レイアウトやデザインを記述）と並ぶ、World Wide Web（WWW）の中核技術の一つです。 JavaScriptの規格には、Ecma Internationalにより標準化されたECMAScript（エクマスクリプト）があります。
（参考：「Wikipedia」）



動的ページ（ユーザの操作や要求に応じて変化）




クライアントサイド処理プログラム（JavaScript）のソースコード




文字列表示（documentオブジェクト）

js_write.html

<html>
<head>
</head>
<body>
<script>
// HTML全体：documentオブジェクト（ブラウザオブジェクトの一つ）
document.write("Hello World!"); // writeメソッドを実行
</script>
</body>
</html>

js_title.html

<html>
<head>
</head>
<body>
<script>
document.title = "test";
// HTML全体：documentオブジェクト
document.write("Hello World!"); // writeメソッドを実行
</script>
</body>
</html>

js_url.html

<html>
<head>
</head>
<body>
<script>
// HTML全体：documentオブジェクト
document.write("Hello World!"); // writeメソッドを実行
document.write("<br>");         // 改行タグ
document.title = "test";        // ページタイトル設定
document.write(document.URL);   // ページのURL表示
</script>
</body>
</html>

反復処理と配列

js_while.html

<html>
<head>
</head>
<body>
<script>
var i=0;
while(i<10){
  i=i+1;
  document.write(i);
  document.write("<br>");
}
</script>
</body>
</html>

js_for.html

<html>
<head>
</head>
<body>
<script>
var a=[1,3,5,7,9,99];
for(var i=0;i<a.length;i++){
  document.write(a[i]);
  document.write("<br>");
}
</script>
</body>
</html>

条件分岐（フィルタ処理）

js_forif.html

<html>
<head>
</head>
<body>
<script>
var a=[1,3,5,7,9,99];
for(var i=0;i<a.length;i++){
  if((a[i]%3)==0){
    document.write(a[i]);
    document.write("<br>");
  }
}
</script>
</body>
</html>
