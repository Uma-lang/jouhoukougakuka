JavaScriptの concat() メソッド

concat() メソッドは、JavaScriptで 配列 や 文字列 を結合するために使用される便利なメソッドです。このメソッドは、元の配列や文字列を変更せずに、新しい配列や文字列を作成します。

配列での concat()

複数の配列を結合し、新しい配列を作成します。元の配列は変更されません。

配列の concat() の基本的な構文

array1.concat(array2, array3, ...);

例:

var array1 = [1, 2, 3];
var array2 = [4, 5, 6];
var newArray = array1.concat(array2);

console.log(newArray);  // [1, 2, 3, 4, 5, 6]
console.log(array1);    // [1, 2, 3]（元の配列は変更されない）

元の配列 array1 と array2 はそのままで、結合された結果の newArray が新しく作成されます。

文字列での concat()

文字列でも、複数の文字列を結合して新しい文字列を作成することができます。

文字列の concat() の基本的な構文

string1.concat(string2, string3, ...);

例:

var string1 = "Hello, ";
var string2 = "World!";
var newString = string1.concat(string2);

console.log(newString);  // "Hello, World!"
console.log(string1);    // "Hello, "（元の文字列は変更されない）

文字列も同様に、元の string1 や string2 は変更されず、新しい文字列 newString が作成されます。

複数の配列や文字列の結合

concat() を使って、複数の配列や文字列を一度に結合することも可能です。

例: 複数の配列の結合

var array1 = [1, 2];
var array2 = [3, 4];
var array3 = [5, 6];
var combinedArray = array1.concat(array2, array3);

console.log(combinedArray);  // [1, 2, 3, 4, 5, 6]

例: 複数の文字列の結合

var str1 = "JavaScript ";
var str2 = "is ";
var str3 = "fun!";
var combinedStr = str1.concat(str2, str3);

console.log(combinedStr);  // "JavaScript is fun!"

利点

  • 元の配列や文字列を変更せずに新しい配列や文字列を作成できるため、安全にデータを操作できます。
  • 複数の配列や文字列を効率よく結合できます。
