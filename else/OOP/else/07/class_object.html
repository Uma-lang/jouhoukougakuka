クラスの定義とオブジェクト生成

クラスとは、オブジェクト指向プログラミングにおける設計図のようなもので、データ（プロパティやメンバ変数）や、データに対して行う操作（メソッド）をまとめたものです。

1. クラスの定義

クラスは以下のように定義します：


class クラス名 {
    public $変数名;  // メンバ変数

    public function __construct($引数) {  // コンストラクタ
        $this->変数名 = $引数;
    }

    public function メソッド名() {  // メソッド
        // 処理
    }
}
  • class: クラスを定義するキーワード。
  • $this: クラス内で、現在のオブジェクトを指します。
  • メンバ変数（プロパティ）: クラス内のデータ。
  • メソッド: クラス内で定義された関数。オブジェクトの操作に使われます。
  • コンストラクタ: オブジェクト生成時に呼ばれる特別なメソッド。初期化に使われます。

2. オブジェクトの生成

クラスを定義した後、そのクラスに基づいてオブジェクトを生成できます：


$オブジェクト名 = new クラス名(引数);

new キーワードを使ってクラスのインスタンス（オブジェクト）を生成します。コンストラクタが呼ばれ、引数が渡されてプロパティが初期化されます。

3. 例: クラスの定義とオブジェクトの生成


<?php
class Car {
    public $color;
    public $model;

    // コンストラクタでプロパティを初期化
    public function __construct($color, $model) {
        $this->color = $color;
        $this->model = $model;
    }

    // 車の情報を表示するメソッド
    public function displayInfo() {
        echo "This car is a " . $this->color . " " . $this->model . "<br>";
    }
}

// オブジェクトを生成して初期化
$myCar = new Car("red", "Toyota");
$myCar->displayInfo();  // メソッドの呼び出し
?>

説明

  • クラス定義: Car クラスが定義され、色とモデルのプロパティを持ちます。
  • コンストラクタ: クラスが初期化される際、渡された引数でプロパティが設定されます。
  • メソッド呼び出し: オブジェクト生成後、displayInfo() メソッドを使って車の情報が表示されます。

4. 実行結果

このコードを実行すると、以下の結果が表示されます：


This car is a red Toyota.

オブジェクト指向プログラミングの利点

  • カプセル化: データとそのデータを操作する機能を1つにまとめます。
  • 再利用性: 一度定義したクラスから、複数のオブジェクトを生成できます。
  • メンテナンス性: クラスごとに役割を分けることで、コードの保守が容易になります。

まとめ

  • クラスはデータとその操作をまとめた設計図です。
  • オブジェクトはクラスから生成され、そのプロパティやメソッドを使用します。
  • クラスのコンストラクタでオブジェクトの初期化を行い、メソッドでその動作を実行します。
